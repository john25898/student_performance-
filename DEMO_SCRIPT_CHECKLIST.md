# Demo Script Checklist

## Pre-Demo Setup

- [ ] Start backend using Python 3.11 environment
- [ ] Confirm dashboard loads at http://127.0.0.1:8000
- [ ] Verify at least one student can be searched in registry
- [ ] Clear browser cache and refresh once

## Demo Flow (7-10 minutes)

- [ ] 1. Open Dashboard: highlight Cohort Shock Detector and Heatmap
- [ ] 2. Run Guided Scenario Runner (Safe case)
- [ ] 3. Run Guided Scenario Runner (Critical case)
- [ ] 4. Show Digital Twin explainability factors
- [ ] 5. Promote an intervention to Workflow Queue
- [ ] 6. Move case through New -> In Progress -> Resolved
- [ ] 7. Show Escalation Radar updates
- [ ] 8. Open Forecaster and run Counterfactual Target Planner
- [ ] 9. Open Reports and explain Before vs After Impact panel
- [ ] 10. Show War Room Mode and urgent case feed

- [ ] Explain escalation policy and accountability
- [ ] Explain measurable operational impact
